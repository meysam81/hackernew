# Screenshots

This directory contains screenshots of HackerNew for documentation and marketing.

## How to Capture Screenshots

### Required Screenshots

1. **hero-dark.png** (1600x900 or similar 16:9)
   - Main feed view in dark mode
   - Show 5-6 stories visible
   - Capture at 2x resolution for retina displays

2. **hero-light.png** (1600x900)
   - Same view in light mode

3. **comments-dark.png** (1600x900)
   - A story detail page with threaded comments
   - Show 3-4 levels of nesting

4. **keyboard-shortcuts.png** (1200x800)
   - Feed view with keyboard hints visible
   - Consider using browser dev tools to highlight the shortcuts

5. **mobile-dark.png** (390x844 - iPhone 14 Pro)
   - Mobile responsive view
   - Dark mode

6. **mobile-light.png** (390x844)
   - Mobile responsive view
   - Light mode

### Capture Tips

1. **Use a clean browser profile**
   - No extensions visible
   - Clear browsing data
   - Standard browser chrome

2. **Recommended tools**
   - macOS: Built-in Screenshot (Cmd+Shift+4)
   - Chrome: DevTools > Device toolbar for mobile
   - [CleanShot X](https://cleanshot.com) for annotations

3. **Resolution**
   - Capture at 2x for retina displays
   - Export at 1x for web use
   - Keep originals in case you need to re-export

4. **File format**
   - PNG for screenshots with text
   - JPEG for photos (if any)
   - Optimize with [ImageOptim](https://imageoptim.com) or similar

### Screenshot Checklist

Before capturing:
- [ ] Clear browser cache and cookies
- [ ] Use incognito/private window
- [ ] Set viewport to consistent size
- [ ] Ensure good content is visible (avoid placeholder text)
- [ ] Check dark/light mode is correctly set
- [ ] Hide bookmarks bar
