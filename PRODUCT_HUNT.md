## Description

### Short Description (260 characters)
```
A modern, open-source reimagining of Hacker News. Built with Astro, Vue 3, and Tailwind CSS. Features dark mode, vim-style keyboard shortcuts, bookmarks, reading history, and a clean minimal interface. A love letter to the HN community.
```

## Responses to Common Questions

### "Why not just use the official HN site?"

```
Great question! You absolutely should‚Äîwe use it every day. HackerNew isn't meant to replace HN. It's a learning project and design exploration. We wanted to see what it would feel like to build HN with modern tools, and share what we learned along the way.
```

### "What makes this different from other HN clients?"

```
A few things:

1. It's a web app, not a native app‚Äîworks everywhere
2. Fully open source with a permissive license
3. Built with modern tools (Astro, Vue 3, Tailwind v4) so others can learn from the code
4. Keyboard-first navigation (vim-style)
5. Optional auth‚Äîworks great without any account

But honestly, use whatever works for you!
```

### "Will you add [feature]?"

```
Maybe! We're keeping the scope intentional‚Äîwe don't want to add features just because we can. But we're definitely open to ideas that would genuinely improve the reading experience.

Feel free to open an issue on GitHub: https://github.com/meysam81/hackernew/issues
```

### "Is this affiliated with Y Combinator or Hacker News?"

```
No, this is an independent open source project. We use the official, public HN API. HackerNew is a fan project‚Äîa love letter to the community we enjoy being part of.
```
